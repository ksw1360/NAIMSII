NAIMS2 기본 사용 가이드

로그인 및 자료분석 메뉴 실행 방법

  1. 바탕화면에서 NAIMS2 아이콘을 찾아 더블 클릭하여 프로그램을 실행합니다.
    프로그램이 시작되면 로그인 창이 나타납니다.
  2. 로그인 창이 나타나면 다음 정보를 입력합니다:
    • 아이디 (ID): user
    • 비밀번호 (Password): 1234
    입력 후 로그인 버튼을 클릭합니다.
    정상적으로 로그인되면 프로그램의 메인 화면으로 이동합니다.
  3. 프로그램 메인 화면 상단의 메뉴 중 자료분석 메뉴를 찾아 클릭합니다.
    자료분석 관련 기능 화면으로 이동합니다.

추가 정보

만약 로그인에 문제가 있거나 메뉴를 찾을 수 없는 경우, 시스템 관리자에게 문의하십시오.
